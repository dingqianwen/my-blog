### 取指定标题

```javascript
let text = `${上面内容}`
console.log(/(?<=title:)[^].+?(?=\n)/.exec(text))
``` 

获取内容如下:

```javascript
Java
```

### 取指定内容：

```javascript
let text = `${上面内容}`
console.log(/(?<=# 目录)[^]*?(?=<Comment><\/Comment>)/.exec(text))
```
